# Beitragen zu Barcode → Bring! Einkaufsliste

Vielen Dank für dein Interesse an diesem Projekt! Beiträge sind willkommen.

## Fehler melden

- Öffne ein [Issue](https://github.com/Noack1978/Barcodescan-zu-Bring-/issues/new?template=bug_report.md)
- Beschreibe den Fehler so genau wie möglich
- Füge die HA-Version, die Integrationsversion und relevante Log-Einträge hinzu

## Verbesserungen vorschlagen

- Öffne ein [Issue](https://github.com/Noack1978/Barcodescan-zu-Bring-/issues/new?template=feature_request.md)
- Beschreibe die gewünschte Funktion und den Anwendungsfall

## Pull Requests

1. Fork des Repos erstellen
2. Einen neuen Branch anlegen (`git checkout -b feature/mein-feature`)
3. Änderungen committen (`git commit -m 'Feature: Beschreibung'`)
4. Branch pushen (`git push origin feature/mein-feature`)
5. Pull Request öffnen

## Entwicklungsumgebung

- Home Assistant (aktuellste Version)
- Python 3.12+
- Die Integration liegt unter `custom_components/barcode_bring/`

## Hinweise

- Bitte keine Breaking Changes ohne vorherige Diskussion im Issue
- Code sollte kompatibel mit der aktuellen HA-Version sein
- Neue Features bitte mit sinnvollen Standardwerten ausstatten
