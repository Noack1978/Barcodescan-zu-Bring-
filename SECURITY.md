# Security Policy

## Unterstützte Versionen

| Version | Unterstützt |
|---------|-------------|
| 2.x     | ✅ Ja        |
| 1.x     | ❌ Nein      |

## Sicherheitslücken melden

Bitte melde Sicherheitslücken **nicht** öffentlich über Issues.

Sende stattdessen eine Beschreibung der Lücke direkt über
[GitHub Security Advisories](https://github.com/Noack1978/Barcodescan-zu-Bring-/security/advisories/new).

Bitte folgende Informationen angeben:

- Beschreibung der Sicherheitslücke
- Schritte zur Reproduktion
- Mögliche Auswirkungen
- HA-Version und Integrationsversion

Ich werde die Meldung so schnell wie möglich bearbeiten.
